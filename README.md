## 🛠️ Requirements

- 🐧 Arch Linux distribution (e.g. Arch, Artix,)
- ☁️ Internet connection for cloning repositories
- 🌱 Having git installed

## 📦 Installation
```
git clone https://github.com/Tapi-Mandy/guhwizard && cd guhwizard && chmod +x guhwizard.sh && ./guhwizard.sh
```
**It builds directly from [Tapi-Mandy/guhwm](https://github.com/Tapi-Mandy/guhwm), and installs:**
* Base packages/Dependencies
* AUR helpers
* Applications
* Configurations

And more.

---
> [!WARNING]
> Has not yet been tested on Artix Linux, or any other Arch derivatives.
